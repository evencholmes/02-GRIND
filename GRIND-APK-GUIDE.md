# GRIND — APK Build Guide
## From single HTML file → Android APK

You already did this for KAZU. Same process, different folder.

---

## STEP 1 — Create your project folder

Make a new folder anywhere on your machine:

```
E:\01 Aims\09 app dev\02 GRIND-HabitOS\grind\
```

---

## STEP 2 — Copy these files into the folder

Copy everything from this package into that folder:

```
grind\
  capacitor.config.json   ← already configured for GRIND
  package.json            ← already configured
  manifest.json           ← PWA manifest
  sw.js                   ← service worker (offline support)
  index.html              ← YOUR GRIND APP FILE (rename grind.html → index.html)
  icon-192.png            ← your app icon (portrait image, resized to 192×192)
  icon-512.png            ← your app icon (portrait image, resized to 512×512)
```

**For icons:** take your portrait GRIND icon image and save two copies:
- `icon-192.png` at 192×192 px
- `icon-512.png` at 512×512 px

You can resize free at: https://www.iloveimg.com/resize-image

---

## STEP 3 — Create the www folder

Capacitor looks for your web files in a `www` subfolder.

Create:
```
grind\www\
```

Copy `index.html`, `manifest.json`, `sw.js`, `icon-192.png`, `icon-512.png`
into the `www` folder too.

So you'll have copies in both:
- `grind\index.html` (root — for PWA in browser)
- `grind\www\index.html` (www — for Capacitor/APK)

---

## STEP 4 — Install dependencies

Open a terminal/command prompt in your `grind\` folder:

```bash
cd "E:\01 Aims\09 app dev\02 GRIND-HabitOS\grind"
npm install
```

This downloads Capacitor into `node_modules\`. Takes 1-2 minutes.

---

## STEP 5 — Add Android platform

```bash
npx cap add android
```

This creates the `android\` folder — the Android Studio project.
Takes a minute. You'll see a lot of output, that's normal.

---

## STEP 6 — Sync your web files into Android

Every time you update `index.html`, run this:

```bash
npx cap sync android
```

This copies your `www\` files into the Android project.

---

## STEP 7 — Add splash screen image

Your splash screen (the landscape/wide image) needs to go here:

```
grind\android\app\src\main\res\drawable\splash.png
```

Recommended size: **2732×2732 px** (square, Android will crop for different screens).
Use your landscape splash image, center the logo, export as PNG.

---

## STEP 8 — Open in Android Studio

```bash
npx cap open android
```

Android Studio opens automatically with your project.

---

## STEP 9 — Build the APK in Android Studio

1. Wait for Gradle sync to finish (bottom bar shows progress)
2. Menu → **Build** → **Build Bundle(s) / APK(s)** → **Build APK(s)**
3. Wait ~2-3 minutes
4. A popup appears: "APK(s) generated" → click **locate**

Your APK is at:
```
grind\android\app\build\outputs\apk\debug\app-debug.apk
```

Transfer to your phone and install. Done.

---

## STEP 10 — For Play Store (when you're ready)

Build a signed release APK instead:
- Menu → Build → Generate Signed Bundle / APK
- Create a keystore (keep this file SAFE — you need it for every update)
- Choose APK → Release
- This gives you `app-release.apk` — the one Play Store accepts

---

## UPDATING THE APP

When you fix bugs or add features:

1. Update `grind\www\index.html` with the new version
2. Run `npx cap sync android`
3. In Android Studio: Build → Build APK(s)
4. New APK ready

## FOLDER STRUCTURE (final)

```
grind\
  capacitor.config.json
  package.json

  manifest.json
  sw.js

  icon-192.png

  www\
    index.html
    manifest.json
    sw.js
    icon-192.png
    icon-512.png
  android\                (auto-generated by cap add android)
    app\
      src\main\res\
        drawable\
          splash.png      ← your splash screen goes here
```

## TROUBLESHOOTING

**"npx: command not found"**
→ Install Node.js from https://nodejs.org (LTS version)

**Gradle sync fails in Android Studio**
→ Let it run, it downloads Android SDK components automatically
→ If it asks to update Gradle, say yes

**App shows white screen on phone**
→ Make sure `www\index.html` exists and has your content
→ Run `npx cap sync android` again

**Icons look blurry**
→ Make sure icon-192.png is exactly 192×192 px, not upscaled

**localStorage not working on APK**
→ It works fine — GRIND uses localStorage + IndexedDB, both work in Capacitor WebView
